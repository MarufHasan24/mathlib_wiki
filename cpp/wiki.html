Mathlib-cpp: wiki

Overview

Mathlib-cpp is an open source math library for C++. It is a collection of functions and constants that are usually used in math. It is a free and open source project.

Installation

Mathlib-cpp is a header only library. You can download the header file from GitHub and include it in your project.
git clone https://github.com/MarufHasan24/mathlib-cpp.git

You can also install it from source code as zip or tar.gz
Source code (Zip) or Source code (tar.gz)

Usage

Mathlib-cpp provides various functions and constants that are usually used in math. It is a free and open source project. It is written in C++.
